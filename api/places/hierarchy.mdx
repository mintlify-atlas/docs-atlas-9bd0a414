---
title: 'Query Hierarchical Place Relationships'
api: 'GET https://input.cbdb.fas.harvard.edu/api/place_belongs_to'
---

Query all places that belong to a specified place in the hierarchical structure.

<Note>
  **Important**: The result always includes the place itself, regardless of whether it has child places.
</Note>

## Parameters

<ParamField query="id" type="number" required>
  Place code (place ID) to query
</ParamField>

<ParamField query="start" type="number">
  Starting record number for pagination
</ParamField>

<ParamField query="list" type="number">
  Number of records to return (page size)
</ParamField>

## Response

<ResponseField name="total" type="number">
  Total number of records
</ResponseField>

<ResponseField name="start" type="number">
  Starting record number of current page
</ResponseField>

<ResponseField name="end" type="number">
  Ending record number of current page
</ResponseField>

<ResponseField name="data" type="array">
  Array of place objects
  
  <ResponseField name="pId" type="number">
    Place ID (place code)
  </ResponseField>
  
  <ResponseField name="pName" type="string">
    Place name in English
  </ResponseField>
  
  <ResponseField name="pNameChn" type="string">
    Place name in Chinese
  </ResponseField>
  
  <ResponseField name="pStartTime" type="number">
    Start year of place's existence
  </ResponseField>
  
  <ResponseField name="pEndTime" type="number">
    End year of place's existence
  </ResponseField>
  
  <ResponseField name="pBName" type="string">
    Parent place name in English
  </ResponseField>
  
  <ResponseField name="pBNameChn" type="string">
    Parent place name in Chinese
  </ResponseField>
</ResponseField>

## Example

### Query Request

Search for all places belonging to "安西都護府" (Anxi Protectorate, ID: 16773):

```bash
curl "https://input.cbdb.fas.harvard.edu/api/place_belongs_to?id=16773"
```

## Notes

- Results are sorted by place ID to ensure consistent ordering
- The queried place itself is always included in the results
- Results follow the same structure as the place search endpoint
