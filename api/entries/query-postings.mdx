## Description

This endpoint queries people based on their entry paths (ways of entering government service), with optional filters for location, date ranges, and coordinates.

<Note>
  This endpoint uses POST method with `Content-Type: application/json`.
</Note>

## Request Body

<ParamField body="entry" type="array" required>
  要查詢的入仕途徑 ID 列表 (Array of entry path IDs to query)
  
  Example: `[36]` for "科舉：進士（籠統）" (Examination: jinshi general)
</ParamField>

<ParamField body="usePeoplePlace" type="number">
  是否啟用與人物相關的地點條件
  
  - `1` - 啟用地點篩選 (Enable location filter)
  - `0` - 不啟用 (Disable, default)
  
  <Warning>
    當此參數為 `0` 時，`peoplePlace` 和 `locationType` 參數將被忽略。
  </Warning>
</ParamField>

<ParamField body="peoplePlace" type="array">
  與人物相關的地點列表 (Array of location IDs)
  
  Only used when `usePeoplePlace=1`.
</ParamField>

<ParamField body="locationType" type="string">
  與人物相關的地點類型
  
  - `pAddr` - 僅查找人物地點 (Person's location only)
  - `eAddr` - 僅查找入仕地點 (Entry location only)
  - `peAddr` - 同時查找兩者 (Both locations)
  
  Only used when `usePeoplePlace=1`.
</ParamField>

<ParamField body="useDate" type="number">
  是否採用年份條件
  
  - `1` - 啟用日期篩選 (Enable date filter)
  - `0` - 不啟用 (Disable, default)
  
  <Warning>
    當此參數為 `0` 時，所有日期相關參數（`dateType`, `dateStartTime`, `dateEndTime`, `dynStart`, `dynEnd`）將被忽略。
  </Warning>
</ParamField>

<ParamField body="dateType" type="string">
  年份類型
  
  - `entry` - 入仕年 (Entry year)
  - `index` - 指數年 (Index year)
  - `dynasty` - 朝代 (Dynasty)
  
  Only used when `useDate=1`.
  
  <Info>
    - 當 `dateType` 為 `entry` 或 `index` 時，僅使用 `dateStartTime` 和 `dateEndTime`
    - 當 `dateType` 為 `dynasty` 時，僅使用 `dynStart` 和 `dynEnd`
  </Info>
</ParamField>

<ParamField body="dateStartTime" type="number">
  年份開始日期 (Start year)
  
  Used when `useDate=1` and `dateType` is `entry` or `index`.
</ParamField>

<ParamField body="dateEndTime" type="number">
  年份結束日期 (End year)
  
  Used when `useDate=1` and `dateType` is `entry` or `index`.
</ParamField>

<ParamField body="dynStart" type="number">
  開始朝代 (Starting dynasty code)
  
  Used when `useDate=1` and `dateType` is `dynasty`.
</ParamField>

<ParamField body="dynEnd" type="number">
  結束朝代 (Ending dynasty code)
  
  Used when `useDate=1` and `dateType` is `dynasty`.
</ParamField>

<ParamField body="useXy" type="number">
  是否使用 xy 座標
  
  - `1` - 返回座標資訊 (Include coordinates)
  - `0` - 不返回 (Exclude coordinates, default)
</ParamField>

<ParamField body="start" type="number">
  結果開始筆數 (Starting record number)
  
  Default: `0`
</ParamField>

<ParamField body="list" type="number">
  列表長度 (Number of records to return)
  
  Default: `10`
</ParamField>

<RequestExample>
```bash cURL
# Query people who entered through jinshi examination between 1368-1644
curl -X POST "https://input.cbdb.fas.harvard.edu/api/query_entry_postings" \
  -H "Content-Type: application/json" \
  -d '{
    "entry": [36],
    "usePeoplePlace": 0,
    "peoplePlace": [],
    "locationType": "peAddr",
    "useDate": 1,
    "dateType": "entry",
    "dateStartTime": 1368,
    "dateEndTime": 1644,
    "dynStart": null,
    "dynEnd": null,
    "useXy": 1,
    "start": 1,
    "list": 10
  }'
```
