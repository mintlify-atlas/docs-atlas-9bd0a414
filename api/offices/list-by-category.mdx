---
title: 'List Offices by Category'
api: 'GET /api/post_list'
description: 'Retrieve offices within a specific category code'
---

Retrieve a list of offices belonging to a specific office category code. This endpoint supports hierarchical category codes and pagination.

## Base URL

All API requests should be made to: `https://input.cbdb.fas.harvard.edu`

## Request Parameters

<ParamField query="id" type="string" required>
  Office category code (e.g., "06" for Tang Dynasty, "0601" for Tang Dynasty - Imperial System)
</ParamField>

<ParamField query="start" type="number">
  Starting record index (0-based). Defaults to 0.
</ParamField>

<ParamField query="list" type="number">
  Number of records to return. If omitted, returns all records in the category.
</ParamField>

## Response Structure

<ResponseField name="total" type="number">
  Total number of records
</ResponseField>

<ResponseField name="start" type="number">
  Starting record index of current response
</ResponseField>

<ResponseField name="end" type="number">
  Ending record index of current response
</ResponseField>

<ResponseField name="data" type="array">
  Array of office records
  
  <ResponseField name="pId" type="string">
    Office ID (office_id)
  </ResponseField>
  
  <ResponseField name="pName" type="string">
    Office name in English
  </ResponseField>
  
  <ResponseField name="pNameChn" type="string">
    Office name in Chinese
  </ResponseField>
  
  <ResponseField name="pNameChnAlt" type="string">
    Alternative Chinese names for the office, separated by semicolons. Corresponds to `[OFFICE_CODES].[c_office_chn_alt]`
  </ResponseField>
</ResponseField>

## Examples

### Get first 100 Tang Dynasty offices

<CodeGroup>

```bash curl
curl "https://input.cbdb.fas.harvard.edu/api/post_list?id=06&start=0&list=100"
```

```javascript JavaScript
fetch('https://input.cbdb.fas.harvard.edu/api/post_list?id=06&start=0&list=100')
  .then(response => response.json())
  .then(data => console.log(data));
```

```python Python
import requests

response = requests.get(
    'https://input.cbdb.fas.harvard.edu/api/post_list',
    params={'id': '06', 'start': 0, 'list': 100}
)
data = response.json()
```

</CodeGroup>

This retrieves offices 0-99 (start=0, end=99, length=100) from the Tang Dynasty category.
