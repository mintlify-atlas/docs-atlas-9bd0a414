## Overview

This endpoint allows you to query social associations (relationships) between people. You can filter by association type, location, index year, and geographic scope. The query finds people (X) who match your criteria and returns their associates (Y) who have the specified relationship.

## Endpoint

```
POST https://input.cbdb.fas.harvard.edu/api/query_associates
```

## Request Parameters

<ParamField body="association" type="array" required>
  Array of association type IDs to query
  
  For example: `[22]` for "Student of" relationship
</ParamField>

<ParamField body="place" type="array">
  Array of location IDs to filter people by
</ParamField>

<ParamField body="usePeoplePlace" type="number" default="0">
  Whether to enable location filtering
  - `1`: Enable location filter (use `place` parameter)
  - `0`: Disable location filter (ignore `place` parameter)
</ParamField>

<ParamField body="useXy" type="number" default="0">
  Whether to include geographic coordinates in results
  - `1`: Include coordinates
  - `0`: Exclude coordinates
</ParamField>

<ParamField body="indexYear" type="number" default="0">
  Whether to filter by index year
  - `1`: Enable index year filter (use `indexStartTime` and `indexEndTime`)
  - `0`: Disable index year filter
</ParamField>

<ParamField body="indexStartTime" type="number">
  Start year for index year filter (only used when `indexYear=1`)
</ParamField>

<ParamField body="indexEndTime" type="number">
  End year for index year filter (only used when `indexYear=1`)
</ParamField>

<ParamField body="broad" type="number" default="0">
  Geographic scope for location matching
  - `1`: Broad scope (±0.06 degrees)
  - `0`: Narrow scope (±0.03 degrees)
</ParamField>

## Request Examples

### Example 1: Find Teachers

Find all teachers (Y) of people (X) who lived in Jianzhou between 960-1250:

```json
{
  "association": [22],
  "place": [101125],
  "usePeoplePlace": 1,
  "useXy": 1,
  "indexYear": 1,
  "indexStartTime": 960,
  "indexEndTime": 1250,
  "broad": 0
}
```

This query finds all people (X) whose index year is between 960-1250 and location is Jianzhou, then returns their teachers (Y).

### Example 2: Find Students

Find all students (Y) of people (X) located in Jianzhou:

```json
{
  "association": [23],
  "place": [101125],
  "usePeoplePlace": 1,
  "useXy": 1,
  "indexYear": 0,
  "indexStartTime": null,
  "indexEndTime": null,
  "broad": 0
}
```

This query finds all people (X) located in Jianzhou, then returns their students (Y).

## Notes

- The `usePeoplePlace` parameter takes priority over the `place` parameter. When `usePeoplePlace=0`, the location filter is disabled regardless of the `place` value.
- Geographic scope (`broad` parameter) controls how precisely locations must match. Use broad scope for historical regions with imprecise boundaries.
- The relationship direction is important: association code `22` means "X is a student of Y", while code `23` means "X is a teacher of Y".
- Use the [Get Association Types](/api/relationships/association-types) endpoint to find available association type IDs.
