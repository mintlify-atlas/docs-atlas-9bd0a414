## Overview

This endpoint allows you to query kinship relationships for a given list of people using either the traditional five-degree kinship system (五服) or custom parameters to control search depth across different relationship types.

## Endpoint

```
POST https://input.cbdb.fas.harvard.edu/api/query_relatives_2
```

## Request Parameters

<ParamField body="people" type="array" required>
  Array of person IDs to query relatives for
</ParamField>

<ParamField body="mCircle" type="number" default="0">
  Whether to use the five-degree kinship system (五服查詢)
  - `1`: Use five-degree system (ignores custom parameters)
  - `0`: Use custom parameters below
</ParamField>

### Custom Search Parameters

These parameters are only used when `mCircle=0`:

<ParamField body="MAncGen" type="number">
  Maximum ancestor distance (generations upward)
</ParamField>

<ParamField body="MDecGen" type="number">
  Maximum descendant distance (generations downward)
</ParamField>

<ParamField body="MColLink" type="number">
  Maximum collateral distance (siblings and cousins)
</ParamField>

<ParamField body="MMarLink" type="number">
  Maximum marriage link distance (in-laws)
</ParamField>

<ParamField body="MLoop" type="number">
  Maximum loop iterations for complex relationship chains
</ParamField>

## Request Example

```json
{
  "people": [1762],
  "mCircle": 0,
  "MAncGen": 1,
  "MDecGen": 1,
  "MColLink": 1,
  "MMarLink": 1,
  "MLoop": 2
}
```

This example queries Wang Anshi's (王安石) relatives with custom parameters: 1 generation up, 1 generation down, 1 degree of collateral relations, and 1 degree of marriage relations.

## Notes

- The `mCircle` parameter takes priority over custom parameters. When `mCircle=1`, the system uses the traditional five-degree kinship system regardless of `MAncGen`, `MDecGen`, `MColLink`, `MMarLink`, and `MLoop` values.
- When the data includes the center person's own record, `rKinship` will be set to `'ego'` and fields starting with `p` will be empty strings.
- The five-degree kinship system (五服) is a traditional Chinese kinship classification system that organizes relatives into five mourning grades based on closeness of relationship.
